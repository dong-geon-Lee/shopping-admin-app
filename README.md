## :mag_right: &nbsp; Intro
> 쇼핑 관리자 페이지 입니다. 제품을 검색하고 페이지를 클릭하여 목록을 확인 할 수 있습니다.  
<br/>

## :link: &nbsp; Link
- https://shopping-admin-app.vercel.app/

> 첫번쨰: 레포지토리를 다운로드 합니다.   
> 두번쨰: 터미널에 npm install 입력합니다.   
> 세번쨰: 터미널에 npm start 입력합니다.
